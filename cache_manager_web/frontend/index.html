Help & Usage Guide

Getting Started

  1. Start the server: uv run python3 cache_manager_web/run.py <agent>
  2. The web UI opens automatically. Select a task from the left panel.
  3. Browse URLs — red = definite issue, yellow = possible issue, green = reviewed.
  4. Use j/k to navigate URLs, Space to toggle screenshot/text.

Chrome Extension Setup

  1. Open chrome://extensions, enable Developer mode.
  2. Click "Load unpacked" and select the cache_manager_web/extension/ folder.
  3. Pin the extension icon for easy access.

Fixing a Single URL

  1. Select the red URL in the list.
  2. Click Recapture Live (or Ctrl+U) — the page opens in a new tab.
  3. If there's a CAPTCHA or login, solve it manually in the browser.
  4. Click the extension icon → Capture This Page.
  5. The UI updates automatically via SSE. The URL turns green.

Batch Recapture

  1. Click Batch Recapture in the toolbar (queues all red URLs).
  2. Click the extension icon → Start Auto-Capture.
  3. The extension opens each URL, waits for load, captures, and advances automatically.
  4. If a CAPTCHA is detected, solve it in the browser tab — the extension waits.
  5. After batch completes, recaptured URLs show as blue (needs human review).

URL Management

  • Flag — manually mark a URL as broken (red). For batch recapture later.
  • Reset — clear all cached content and flag. Use when cache is corrupted.
  • Edit — change the URL link (content moves to the new URL).
  • Add URL — add a new URL (auto-flagged). PDF URLs detected by .pdf suffix.
  • Delete — permanently remove a URL and its files.
  • Upload PDF / MHTML — manually replace content by file upload or drag-and-drop.

Recommended Workflow

  1. Load an agent cache and review the issue count.
  2. Use n to jump through issues. Quick-check screenshot vs text.
  3. For false positives (yellow), just viewing auto-marks as reviewed.
  4. For real issues (red), use Batch Recapture to fix them all at once.
  5. After batch, review blue URLs — press r to confirm each is fixed.
  6. For stubborn pages (anti-bot, login), use Recapture Live one by one.
  7. For PDF pages misrecorded as web, use Upload PDF or let the extension auto-detect.

Tips

  • Drag-and-drop .pdf or .mhtml files onto the preview panel to upload.
  • The extension auto-detects PDF pages and uploads them directly.
  • Use the Issues and Todo filters to focus on unresolved URLs.
  • The Answer view (3) shows agent answer markdown files for the current task.
  • After batch recapture, use n to jump only to remaining issues.
